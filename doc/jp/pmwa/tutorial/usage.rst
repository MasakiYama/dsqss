使用方法
---------------

PMWAは以下の手順で使用します。

1. 格子定義ファイルの作成 (``lattgene_P`` を利用)
2. 入力ファイルの作成
3. PMWAの実行 (ボソン系：``pmwa_B`` ,スピン系：``pmwa_H`` の実行)
4. 出力ファイルの生成

また,1, 2を同時に行うための簡易ツール ``dsqss_pre.py`` も用意されています。
ここでは1-4について使用方法やパラメータについて順に説明します。
